Transesterification¶

class biosteam.units.Transesterification(ID='', outs=(), ins=None, efficiency=None, methanol2lipid=None, T=None, catalyst_molfrac=None)[source]¶

Create a transesterification reactor that converts ‘Lipid’ and ‘Methanol’ to ‘Biodiesel’ and ‘Glycerol’. Finds the amount of catalyst ‘NaOCH3’ required and consumes it to ‘NaOH’ and ‘Methanol’.

Parameters

ID: [str] Unique identification. If set as ‘’, a default ID will be chosen.

outs: tuple[str or Stream] Output streams or IDs to initialize output streams. If None, leave streams missing. If empty, default IDs will be given.

ins: tuple[str or Stream] Input streams or IDs to initialize input streams. If None, leave streams missing. If empty, default IDs will be given.

efficiency: Efficiency of conversion (on a ‘Lipid’ basis)

methanol2lipid: Methanol feed to lipid molar ratio

catalyst_molfrac: Catalyst to methanol molar ratio

T: Operating temperature (K)

ins

[0] Lipid feed

[1] Methanol feed (includes catalyst)

outs

[0] Product
methanol2lipid¶

Methanol feed to lipid molar ratio.

catalyst_molfrac¶

Catalyst molar fraction in methanol feed.
