MixTank¶

class biosteam.units.MixTank(ID='', outs=(), ins=None)[source]¶

Create a mixing tank with volume based on residence time.

References

[1] I.K. Kookos, Preliminary Chemical Process Synthesis and Design, Tziolas Publishing, Thessalonika, Greece, 2008 (book in Greek).
ID¶

Unique Identification (str). If set as ‘’, it will choose a default ID.

diagram(radius=0, file=None, format='svg')¶

Display a Graphviz diagram of the unit and all neighboring units within given radius.

Parameters

radius: [int] Maxium number streams between neighbors.

file: Must be one of the following:
  • [str] File name to save diagram.
  • [None] Display diagram in console.

format: Format of file.
