Unit decorators¶

Unit decorators extend the behavior of Unit subclasses.

design decorator¶

The design decorator adds a design item to the class.

Create a Shredder Unit subclass with mass flow rate as a design item:

In [1]:

from biosteam import *
from biosteam.units.decorators import *

@design('Flow rate', 'kg/hr', lambda self: self.ins[0].massnet)
class Shredder(Unit):
    # Set input stream exactly equal to output stream
    _linkedstreams = True

### Test a Shreadder object ###

Unit.CEPCI = 603.1 # For year 2018

# Set up Stream.species
species = Species()
species.SugarCane = compounds.Substance('SugarCane', MW=1, default_phase = 's')
Stream.species = species

# Display flow rate in kg/hr
Stream.display_units.flow = 'kg/hr'

# Create Shredder object and simulate
shredder = Shredder(ins=Stream(SugarCane=1e6, units='kg/hr'))
shredder.simulate()

shredder.show()
shredder.results()

    flow (kg/hr): SugarCane  1e+06
outs...
[0] d2
    phase: 'l', T: 298.15 K, P: 101325 Pa

cost decorator¶

The cost decorator adds an item purchase cost based on the exponential scale up equation:

\(C_{f.o.b.} = (N)(CE)(cost)\big(\frac{basis}{S}\big)^{exp}\)

\(Electricity\ rate = (N)(kW)\big(\frac{basis}{S}\big)\)

Create a Shredder Unit subclass but this time with a cost item:

In [2]:

@cost('Flow rate', cost=2.5e6, CE=567.3, exp=0.6, S=500e3, kW=3000)
@design('Flow rate', 'kg/hr', lambda self: self.ins[0].massnet)
class Shredder(Unit):
    _linkedstreams = True

### Test a Shreadder object ###

shredder = Shredder(ins=Stream(SugarCane=1e6, units='kg/hr'))
shredder.simulate()

shredder.show()
shredder.results()

ins...
[0] d3
    phase: 'l', T: 298.15 K, P: 101325 Pa
    flow (kg/hr): SugarCane  1e+06
outs...
[0] d4
    phase: 'l', T: 298.15 K, P: 101325 Pa
    flow (kg/hr): SugarCane  1e+06

N decorator¶

The N decorator adds a design item for the number of parellel units needed due to the basis limit.

In [5]:

@N('Flow rate', 'Number of shredders', limit=500e3)
@cost('Flow rate', N='Number of shredders',
      cost=2.5e6, CE=567.3, exp=0.6, S=500e3, kW=3000)
@design('Flow rate', 'kg/hr', lambda self: self.ins[0].massnet)
class Shredder(Unit):
    _linkedstreams = True

### Test a Shreadder object ###

shredder = Shredder(ins=Stream(SugarCane=1e6, units='kg/hr'))
shredder.simulate()

shredder.show()
