class biosteam.utils.UnitManager(clients, *args, **kwargs)¶

Create a UnitManager object for handling units of measure of a list of dictionaries (clients). When an item in UnitManger changes (e.g. unit_manager[‘T’]=’K’ to unit_manager[‘T’]=’degC’), all dictionaries in clients with the same key change values accordingly.

class biosteam.utils.SmartBook(units={}, bounds=None, obj=None, *args, **kwargs)¶

Create a dictionary that represents values with units of measure and alerts when setting an item out of bounds.

Parameters

units: [UnitManager or dict] Dictionary with units of measure.

bounds: [dict] Dictionary with bounds.

obj: [dict] Object which the SmartBook object belongs to.

*args: Key value pairs to initialize.

**kwargs: Key value pairs to initialize.

Examples

SmartBook objects provide an easy way to keep track of units of measure and enforce bounds. When the units change, Pint Quantity objects are used to change the values accordingly.

Create a SmartBook object:

>>> sb = SmartBook(units={'Duty': 'kJ/hr', 'T': 'K'}, bounds={'T': (0, 1000)}, T=350)
>>> sb
{'T': 350 (K)}

Change units:

>>> sb.units['T'] = 'degC'
>>> sb
{'T': 76.85 (degC)}

>>> sb.bounds
{'T': array([-273.15, 726.85])}

Add items and units:

>>> sb['P'] = 101325
>>> sb
{'T': 76.85 (degC),
 'P': 101325}

>>> sb.units['P'] = 'atm'
>>> sb
{'T': 76.85 (degC),
 'P': 101325 (atm)}

Set value that is out of bounds:

>>> sb['T'] = -300
RuntimeWarning: T (-300 degC) is out of bounds. T must be within -273.15 and 726.85 degC.

The units dictionary becomes a UnitManager object with a reference to all dictionaries (clients) it controls.

View the UnitManager:

>>> sb.units
UnitManager:
{'T': 'degC', 'P': 'atm'}

View all clients:

>>> sb.units.clients
[{'T': 76.85 (degC), 'P': 101325 (atm)}, {'T': array([-273.15, 726.85])}]

Inner SmartBook objects are easy to read, and can be made using the same units and bounds.

Create a new SmartBook object:

>>> sb2 = SmartBook(sb.units, sb.bounds, T=25)
>>> sb2
{'T': 25 (degC)}

Set new SmartBook object as an item:

>>> sb['Inner'] = sb2
>>> sb
{'T': 76.85 (degC),
 'P': 101325 (atm),
 'Inner':
    {'T': 25 (degC)}}

class biosteam.utils.WeakRefDict(check_ID=False, **kwargs)¶

Create a WeakRefDict object that stores only weak references. When item getter is used, it returns the call of the weak reference. If the key does not exist, it returns None. When check_ID is True, an IDconflict error will be raised when setting with a key/ID already occupied by an object. To bypass this check, include ‘*’ to the key/ID.

Parameters

check_ID: [bool] if True, no key/ID can be specified twice.

**kwargs: [dict] Key-value pairs to initialize dictionary.

class biosteam.utils.Sink(stream, index)¶

Create a Sink object that connects a stream to a unit using piping notation:

Parameters

stream: [Stream]

index: [int]

Example

First create a stream and a Mixer:

>>> stream = Stream('s1')
>>> unit = Mixer('M1')

Sink objects are created using piping notation:

>>> stream-1
<Sink: s1-1>

Use pipe notation to create a sink and connect the stream:

>>> stream-1-unit
>>> M1.show()

Mixer: M1
ins...
[0] Missing stream
[1] s1
    phase: 'l', T: 298.15 K, P: 101325 Pa
    flows:  0
outs...
[0] d27
    phase: 'l', T: 298.15 K, P: 101325 Pa
    flows:  0
