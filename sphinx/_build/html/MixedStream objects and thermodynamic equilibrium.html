MixedStream objects and thermodynamic equilibrium¶

MixedStream is an extention of Stream with ‘s’ (solid), ‘l’ (liquid), ‘L’ (LIQUID), and ‘g’ (gas) flow rates. The upper case ‘LIQUID’ denotes that it is a distinct phase from ‘liquid’.

Create MixedStream Objects¶

Before initializing MixedStream objects, first set the species:

In [1]:

import biosteam as bs
bs.MixedStream.species = bs.Species('Water', 'Ethanol')

Initialize with an ID and optionally T and P. Then you can set flow rates for different phases:

In [2]:

ms1 = bs.MixedStream(ID='ms1', T=351, P=101325)
ms1.setflow('l', Ethanol=1, units='kmol/hr')
ms1.setflow('g', Ethanol=2) # Assuming kmol/hr
ms1.getflow('l', 'Ethanol', 'Water')

Out[2]:

array([1., 0.])

You can view flow rates differently by setting the units for show.

In [3]:

ms1.show(flow='kg/hr', T='degC')

MixedStream: ms1
 phase: 'lg', T: 77.85 degC, P: 101325 Pa
         species  kg/hr
 liquid: Ethanol  46.1

 vapor:  Ethanol  92.1

Get and set flow rates¶

Flow rates are stored in solid_mol, liquid_mol, LIQUID_mol, and vapor_mol arrays:

In [4]:

ms1.solid_mol

Out[4]:

array([0., 0.])

In [5]:

ms1.liquid_mol

Out[5]:

array([0., 1.])

In [6]:

ms1.LIQUID_mol

Out[6]:

array([0., 0.])

In [7]:

ms1.vapor_mol

Out[7]:

array([0., 2.])

Assign flows using these properties:

In [8]:

ms1.liquid_mol[:] = [2, 1]
ms1.vapor_mol[1] = 3
ms1

MixedStream: ms1
 phase: 'lg', T: 351 K, P: 101325 Pa
         species  kmol/hr
 liquid: Water    2
         Ethanol  1

 vapor:  Ethanol  3

Mass and volumetric flow rates are also availabe as property_array objects

In [9]:

ms1.liquid_mass

Out[9]:

property_array([36.031, 46.068])

In [10]:

ms1.liquid_vol

Out[10]:

property_array([0.037, 0.063])

Assign flows through the mass or vol property arrays:

In [11]:

# Set gas phase specie flow rate by index
ms1.vapor_mass[0] = 10
ms1.show()

MixedStream: ms1
 phase: 'lg', T: 351 K, P: 101325 Pa
         species  kmol/hr
 liquid: Water    2
         Ethanol  1

 vapor:  Water    0.555
         Ethanol  3

In [12]:

# Set liquid phase flow rates assuming same order as in species object
ms1.liquid_vol[:] = [0.1, 0.2]
ms1.show()

MixedStream: ms1
 phase: 'lg', T: 351 K, P: 101325 Pa
         species  kmol/hr
 liquid: Water    5.39
         Ethanol  3.19

 vapor:  Water    0.555
         Ethanol  3

Single phase flow properties¶

Add ‘net’ to get the net flow rate

In [13]:

ms1.liquid_molnet

Out[13]:

8.581060526320748

In [14]:

ms1.vapor_massnet

Out[14]:

array(148.20532, dtype=object)

In [15]:

ms1.solid_volnet

Out[15]:

array(0.0, dtype=object)

Add ‘frac’ to get the composition

In [16]:

ms1.vapor_molfrac

Out[16]:

tuple_array([0.156, 0.844])

In [17]:

ms1.liquid_massfrac

Out[17]:

tuple_array([0.3976676602743484, 0.6023323397256516], dtype=object)

In [18]:

ms1.solid_volfrac

Out[18]:

tuple_array([0.5, 0.5])

Note: When a phase has no flow rate, all specie fractions will be equal.

Vapor-liquid equilibrium¶

Adiabatic and isobaric conditions:

In [25]:

ms2 = bs.MixedStream('ms2', T=370)
# Set flows in same order as in Species object
ms2.setflow('g', (1, 2))
ms2.setflow('l', (2, 1))
ms2.VLE()
ms2

Liquid-liquid equilibrium¶

Initialize with MixedStream object with water and octane:

In [28]:

# Make stream with hydrophobic species
ms3 = bs.MixedStream('ms3', species=bs.Species('Water', 'Octane'))
ms3.setflow('l', (2, 2))
ms3

MixedStream: ms3
 phase: 'l', T: 298.15 K, P: 101325 Pa
         species  kmol/hr
 liquid: Water    2
         Octane   2

Adiabatic and isobaric conditions:

In [29]:

# Must set liquid-LIQUID guess splits
ms3.LLE()
ms3

MixedStream: ms3
 phase: 'lL', T: 298.15 K, P: 101325 Pa
         species  kmol/hr
 liquid: Water    1.98
         Octane   2.49e-05

 LIQUID: Water    0.0167
         Octane   2

Note: LLE assumes no significat temperature change with phase partitioning, resulting in constant temperature.

Isothermal and isobaric conditions:

In [30]:

# Must set liquid-LIQUID guess splits
ms3.LLE(T=340)
ms3

MixedStream: ms3
 phase: 'lL', T: 340 K, P: 101325 Pa
         species  kmol/hr
 liquid: Water    1.94
         Octane   5.37e-05

 LIQUID: Water    0.0627
         Octane   2
